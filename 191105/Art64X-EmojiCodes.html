How to use emojis in web page content design

Emojis are a fun way to communicate information in the web space. Emojis are read as visual media. But emojis are actually defined by code, and are not graphics.

Basic steps of looking up emoji code

  • Google "hex code for [[whatever]] emoji"
  • In search results, locate your emoji
  • Look for the HTML hex code or the HTML decimal code. Copy the code.
  • Paste the code into your HTML. See the examples above.

Example usage of the basic code lookup steps for the smiley face emoji

  • Google "hex code for smiley face emoji" https://www.google.com/search?q=hex+code+for+smiley+face+emoji&rlz=1C1GCEU_enUS874US874&oq=hex+code+for+smiley+face+emoji&aqs=chrome..69i57.4654j0j4&sourceid=chrome&ie=UTF-8
