Hex Codes

Hex codes have six characters:
123456
Position 12 = red value
Position 34 = green
Position 56 blue

Hex code values range from 00-FF: 00 = 0%
33 = 20%
66 = 40%
99 = 60%
CC = 80%
FF = 100%
